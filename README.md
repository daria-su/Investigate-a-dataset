# Investigate-a-dataset #

## Udacity Project in DAND Program - Investigate a Dataset

### Project details:

In this project you will investigate a dataset about "Medical Appointment No Shows" in Brazilian hospitals.

### Core files of the project:

In file investigate-a-dataset-project2.ipynb you can find python jupiter notebook.

There is also html version of this file, saved in investigate-a-dataset-project2.html

### Data sets:

You can find dataset in file noshowappointments-kagglev2-may-2016.csv

Data was taken from site: https://www.kaggle.com/joniarroba/noshowappointments.
